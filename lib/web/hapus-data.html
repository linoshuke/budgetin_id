Petunjuk Penghapusan Data Akun Budgetin

Kami menghargai privasi dan hak Anda atas data pribadi Anda. Jika Anda memutuskan untuk tidak lagi menggunakan layanan "Budgetin", Anda dapat menghapus akun dan seluruh data terkait secara permanen melalui aplikasi.

Langkah-langkah Penghapusan Akun

Untuk menghapus akun Anda, silakan ikuti petunjuk berikut di dalam aplikasi Budgetin:

  1. Buka aplikasi Budgetin di perangkat Anda.
  2. Masuk ke halaman "Pengaturan" (biasanya diwakili oleh ikon roda gigi).
  3. Di dalam menu Pengaturan, cari dan pilih bagian "Zona Berbahaya" atau "Akun & Keamanan".
  4. Tekan tombol "Hapus Akun". Tombol ini biasanya berwarna merah untuk menandakan tindakan yang bersifat permanen.
  5. Anda akan diminta untuk melakukan konfirmasi terakhir. Harap baca peringatan yang ditampilkan dengan saksama.
  6. Setelah konfirmasi, akun dan seluruh data Anda akan dijadwalkan untuk penghapusan permanen dari sistem kami.

Data Apa yang Akan Dihapus?

Saat Anda menghapus akun, data berikut akan dihapus secara permanen:

  • Profil akun Anda (nama, email, foto profil).
  • Seluruh riwayat transaksi (pemasukan dan pengeluaran).
  • Data anggaran (budget) yang telah Anda buat.
  • Data target tabungan (saving goals).
  • File atau gambar terkait transaksi yang mungkin pernah Anda unggah.
Penting: Proses penghapusan akun bersifat permanen dan tidak dapat dibatalkan. Setelah data dihapus, kami tidak dapat memulihkannya kembali.

Jika Anda mengalami kesulitan atau memiliki pertanyaan, jangan ragu untuk menghubungi tim dukungan kami di alamat email dukungan@Budgetin.com (ganti dengan email Anda).
